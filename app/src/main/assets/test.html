Java Interface 可用于接口测试

  • Toast 提示
    代码：
    AndroidNative.hello();

  • Toast 提示带参数
    代码：
    AndroidNative.hello('Hello Android');

  • 获取 Android 返回值
    代码：
    AndroidNative.getAndroid();

  • Android Js 方法调用打印时间差
    代码：
    AndroidNative.getAndroidTime();
